# 全能解题助手（Gemini 3 多模态）

前端技术栈：Vite + React + TypeScript + Tailwind CSS。

## 目录结构（核心）

- `src/main.tsx`：应用入口
- `src/App.tsx`：主页面与状态流转
- `src/components/`：UI 组件（解析面板、加载页等）
- `src/services/`：业务服务（Gemini 调用）
- `src/types.ts`：类型定义

## 本地运行

**前置条件：** Node.js

1. 安装依赖：`npm install`
2. 配置环境变量：在 `.env.local` 中设置：`VITE_GEMINI_API_KEY=你的 Gemini API Key`
3. 启动开发服务器：`npm run dev`

## 构建与预览

- 构建：`npm run build`
- 本地预览：`npm run preview`
